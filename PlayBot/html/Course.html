Our Course

Explore the PlayBot Curriculum designed to engage children in robot programming through play.

Introduction to Robot Programming

This course introduces children to the basics of robot programming using fun and interactive activities.

Advanced Robot Programming

Building on the basics, this course covers more advanced programming concepts and challenges.

Robot Programming Projects

Children will work on projects that apply their programming skills to solve real-world problems.

我們的課程

探索 PlayBot 課程，旨在通過遊戲讓孩子們參與機械人編程。

機械人編程入門

本課程通過有趣和互動的活動向孩子們介紹機械人編程的基礎知識。

高級機械人編程

在基礎知識的基礎上，本課程涵蓋更高級的編程概念和挑戰。

機械人編程項目

孩子們將參與項目，應用他們的編程技能來解決現實世界的問題。
